- 解决问题    
(i)裁剪的区域可能没法包含物体的整体；(ii)变形操作造成目标无用的几何失真。如果识别目标尺寸变化多样，那么提前定义好的尺寸就可能不太适合。
- 达成目标    
多尺寸输入均可, **注意:batch内需要保持尺寸一致**
